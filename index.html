What is grid?

Grids are two-dimenisioinal layouts that allow you to place objects verticallly and horizontally and both simultsniously. Grid is highly flexible and responsive making it a powerful tool for creating complex and responsive websites.

Grid-template-rows

Defines the sizes and structure of rows in a grid container. Allows you to specify the height of individual rows or the overall structure of rows within the grid.

Grid-template-columns

Defies the sizes and structures of columns in a grid container. Allows you to specify the width of individual columns or the overall structure of columns within the grid.

Grid functions

You can define the rows and columns using fixed sizes (Ex:100px), Relative units like percentages to create responsive layouts (Ex: 20%), or using Auto to allow the height to automatically adjust for content. (Ex:auto), The repeat function simpllifies the process of defining the columns and rows with the same size (Ex: repeat(3, 100px)
